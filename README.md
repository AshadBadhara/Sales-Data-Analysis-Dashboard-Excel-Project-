# Sales-Data-Analysis-Dashboard-Excel-Project-
Developed a business-ready sales performance dashboard using Excel by analyzing ~9,900 transactional records. Identified high-performing categories, profit-draining products, and top revenue states through KPI-driven analysis and visual storytelling. This project showcases practical data analytics skills including data cleaning, exploratory analysis, pivot-based reporting, and insight generation for business stakeholders.
